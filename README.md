# Simon-Game

## Description

Simon-Game is a web-based implementation of the classic memory game "Simon." The game challenges players to remember and replicate a sequence of colored buttons. Test your memory and pattern recognition skills with this fun and interactive game!

## How to Play

1. Open the game in your web browser.
2. Press any key to start the game.
3. Simon will show you a sequence of colored buttons by lighting them up.
4. Your task is to repeat the exact sequence by clicking on the buttons in the correct order.
5. As you progress, the sequences will get longer and more challenging.
6. If you make a mistake, the game will end, and you can start again by pressing any key.
